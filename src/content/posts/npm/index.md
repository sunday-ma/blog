**npm**是世界上最大的**软件库**（Registry）

**npm**也是一个软件包**管理器**和**安装器**

## 世界上最大的软件注册中心（图书馆）

- **npm**是世界上最大的**软件注册中心**。
- 该注册表包含超过 800,000 个**代码包**。
- **开源**开发人员使用**npm**来**共享**软件。
- 许多组织还使用 npm 来管理私有开发。

## 使用 npm 是免费的

- **npm**是免费使用的。
- 无需注册、登录，即可下载所有npm公共软件包。
